# Matrix Diagonal Sum Calculator

Простой консольный проект на C# для вычисления суммы главной или побочной диагонали квадратной матрицы.

---

## English

### Description

This is my first GitHub project created while learning C#.  
The program allows the user to:

- Create a square matrix manually or fill it with random numbers  
- Choose between the main diagonal or secondary diagonal  
- Calculate and display the diagonal sum  
- Repeat calculations if needed  

The project demonstrates working with multidimensional arrays, method overloading, input validation, random number generation, loops, and basic console interaction.

### Features

- Manual matrix input  
- Random matrix generation  
- Diagonal selection  
- Input validation  
- Repeat mode  

### Technologies

- C#  
- .NET  
- Console application  

### How to Run

1. Clone the repository:  
git clone https://github.com/AuroraMondegard/MatrixDiagonals.git
2. Open the project in Visual Studio  
3. Run (Ctrl + F5)
